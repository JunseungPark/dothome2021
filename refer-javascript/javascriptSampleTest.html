세수중 최솟값

Q : 100 이하의 자연수 A, B, C 를 입력받아 세 수 중 가장 작은 값을 출력하는 프로그램을 작성하세요.

입력 설명
첫 번째 줄에 100이하의 세 자연수가 입력된다.
출력 설명
첫 번째 줄에 가장 작은 수 를 출력 한다.
입력 예제
10, 50 , 85
출력 예제
10

let arr = [10, 50 , 85];
let min = arr[0];
for(let i = 1; i < arr.length; i++) {
    if (min > arr[i]) {
        min = arr[i];
    }
} 
document.write(min);

최솟값 구하기

Q : 7개의 숫자가 주어지면 그숫자중 가장 적은 수를 출력하는 프로그램을 작성하시요!

입력 설명
첫 번째 줄에 100이하의 7개의 자연수가 입력된다.
출력 설명
첫 번째 줄에 가장 작은 수 를 출력 한다.
입력 예제
5, 3, 7, 11, 2, 15, 17
출력 예제
2, 3, 5, 7, 11, 15,17

let arr1 = [5, 3, 7, 11, 2, 15, 17];

for (let i = 0; i < arr1.length; i++) {
    let swap;
    for (let j = 0; j < arr1.length - 1 - i; j++) {
        if (arr1[j] > arr1[j + 1]) {
            swap = arr1[j];
            arr1[j] = arr1[j + 1];
            arr1[j + 1] = swap;
        }
    }
}

document.write(arr1);
